NGraphX

Your knowledge network

Welcome to NGraphX

NGraphX is a node editor optimized for quick interaction which allows you to map out any information you wish in a hierarchical manner. By laying out nodes in hierarchies you create an editable, navigable map of knowledge that can be searched for different criteria of your nodes and their content.Although being run in a webbrowser NGraphX works offline, no data is being sent anywhere. Your data resides inside your browser's local storage. Your data is yours: no strings attached!

User guide

Basics

There are a couple of basic concepts you need to understand in order to work with NGraphX.

  • Every Network (e.g. collection of nodes) has a root node
  • The root node can not be deleted
  • The root node can be renamed
  • Every node needs to have exactly one parent
  • A node can have as many children as you wish
  • Nodes can be collapsed to hide their children

"Network"

A collection of nodes is what is called a network in NGraphX. A network is always parented below a "root" node. You can save your network by pressing "Save Network" in the toolbar. The file that is offered for download when you press the save button, will always have the name of your "root" node. Usually browsers increment the filename when the file already exists:
rootName.json -> rootName(1).json -> rootName(2).json
This makes it easy to keep incremental backups of your node network - which is very recommended!
NGraphX is beta software. Save often!

Where is my data?

Your data is inside of your browser's local storage!
That is pretty good except for one thing: you have to take care of it!
Make sure to save your Network file before deleting browser data like cookies / storage etc. The local storage is a special storage area of your webbrowser that websites can use on your computer. Every website can only access the local storage for it's own URL. Read up on the local storage here.

UX

The User Experience Design for NGraphX is based on the concept of quick interaction and intelligent switching of focus inside the software. When you add a node for instance, the first thing you'll want to - always! - do, is to give the node a name, that is why it is automatically triggered to give focus to the Node Name input field after adding a node. No need to touch the mouse! With focus inside the Graph Editor press Enter or Shift+LMB somewhere in the empty and start typing as focus is switched to the Node Editor.

NGraphX's UI consists of three parts. You will use all three of them a lot and you can navigate either with the keyboard or with the mouse. Keyboard only use makes the use of NGraphX extremely quick!

UI - Search and Toolbar

Search and Toolbar. The Search Input field can be reached by pressing s in the Graph Editor

TAB

 Moves focus from search field to search options to Go Button and then through second row buttons

Enter

 (inside the search input) Starts search

Escape

 Focuses the Node Graph

The search and toolbar allows you to search your network and provides several functions in the form of buttons. As with all areas of NGraphX, the search and toolbar will be higlighted yellow if your focus is inside of it. Entering a search term and pressing Enter will start the search as will pressing the big yellow Go button.

Clicking on "Everything" will open a drop down menu allowing you to restrict your search to either names, tags, content or not to restrict it and search for the term entered in everything at once. Searching for "he" will find "hello" "heaven" "he said" and so forth. Search results will be displayed to you in the lower part of the node editor.

To quickly change the search modes, Tab until focus is on the Search Mode field and then press Arrow Up or Arrow Down.

UI - Buttons

  • Add Node/Add Child - Adds a new node to the network
  • Delete Node - Removes the currently selected node
  • Load Network - Choose a file to load
  • Save Network - Save the current network to a file on your disk
  • Save Online - Not yet implemented / will allow you to save on a webserver
  • Toggle Editor - Toggles the node editor on/off to maximize the graph area
  • Help - Opens this documentation

UI - Node Editor

TAB

 Moves focus from Node's name to tags to content and back to the graph editor

Enter

 (With focus on the Tag "+" button) creates an input field to enter a tag

Ctrl + Double click

 Removes the clicked tag

The node editor gets activated as soon as you select a node in the graph editor. Once activated you can edit the contents of your nodes inside this editor. Changing the node's name or any data inside this editor will toggle an automatic save of your network to the local storage.

Change the nodename

Simply click inside "Node" input field and enter the new name. If you wish to cancel your change, press escape. If you wish to accept the change, leave the input field or press enter - it will get saved both ways.

Adding or removing tags

To add a tag click inside the orange "+" button or press enter when it has focus! An input will appear and you can enter the tag you wish to give this node. To remove a tag hold Ctrl and perform a double click on the tag.

Editing a tag

Editing a tag means changing it from "recipE" to "recipe" for instance. If you edit a tag the tag will be changed on all nodes that use this tag! All nodes using "recipE" as a tag will now use "recipe" instead! Editing a tag is only a good idea if you misspelled something or you want to recategorize your data, for instance from "recipe" to "dish".

Content

Nodes can have content attached to them. Click inside the textfield and enter any information you wish. On exiting the field the information will be saved.

Adding a node

To add a node the quick way make sure the graph editor has focus and press Enter.
You can also hold shift and left click somewhere in the empty space of the graph to add a node.
If you have a node selected the new node will be a child of the selected node! If there is no node selected it will automatically be a child of the root node. After the node has been added focus will automatically switch to the node editor and you will be able to start typing the name.

PRO TIP

When you add a node in the graph editor, focus will automatically switch to the Node name field. After adding a node, simply enter the name. If you want to add more information to this node press tab and you will be on the + button. Pressing Enter now will activate adding a tag. After pressing enter again to confirm your tag, continue typing as another input is generated for the next tag! If you do not need another tag, press tab while leaving the input empty, which will take you to the content field! After entering your content press tab again which will take focus back to the graph editor!
